VeloQ Gmail & Password Control

This page no longer asks for a GitHub token. VeloQ users sign in with Gmail and password; passwords are stored only as PBKDF2 hashes in the signed GitHub manifest.

Add or update a user

On the owner PC, run:

E:\veloq 6 (2nd gen)\VELOQ_GITHUB_ADMIN.cmd

Choose option 1, then enter Gmail, password, expiry, and either Unlimited or a total request limit. The tool signs and publishes the GitHub control automatically.

Change the model

Run the same tool and choose option 4.
